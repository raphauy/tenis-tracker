# New Features

Borradores y notas de features **antes** de tener PRP. Aquí se escribe libre: idea, motivación, mockups, dudas abiertas, capturas de inspiración.

Cuando la feature está suficientemente entendida → se promueve a `docs/PRPs/<feature>-prp.md` y se referencia desde el roadmap.

Convención: `<feature>.md`, opcionalmente con fecha `YYYY-MM-DD` si conviene versionar el borrador. Imágenes en `docs/new-features/images/`.
